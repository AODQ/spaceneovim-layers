# Vimr Layer
This layer is an opinionated setup that works nicely with the [Vimr](https://github.com/qvacua/vimr) editor for Neovim.

#### Table of Contents
- [Install](#install)

## Install
Add the `+gui/vimr` layer in your configuration file,

```viml
function! Layers()
  " ...
  Layer '+gui/vimr'
  " ...
endfunction
```
