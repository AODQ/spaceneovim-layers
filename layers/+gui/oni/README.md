# Oni Layer
This layer is an opinionated setup that works nicely with the [Oni](https://www.onivim.io) editor for Neovim.

#### Table of Contents
- [Install](#install)

## Install
Add the `+gui/oni` layer in your configuration file,

```viml
function! Layers()
  " ...
  Layer '+gui/oni'
  " ...
endfunction
```
