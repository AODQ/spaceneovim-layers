# Start Screen Layer
This layer displays a nice start screen, using

- [mhinz/vim-startify](https://github.com/mhinz/vim-startify).

#### Table of Contents
- [Install](#install)

## Install
Add the `+nav/start-screen` layer in your configuration file,

```viml
function! Layers()
  " ...
  Layer '+nav/start-screen'
  " ...
endfunction
```
